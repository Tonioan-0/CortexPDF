How to Get an API Key

A guide to finding or creating API keys for CortexPDF supported AI providers.

psychology

Google Gemini

  1. Go to Google AI Studio at aistudio.google.com/app/apikey.
  2. Sign in with your Google account.
  3. Click on the "Create API key" button.
  4. Copy the generated key and paste it into CortexPDF.
chat

OpenAI (ChatGPT)

  1. Go to the OpenAI Platform at platform.openai.com/api-keys.
  2. Log in or create an account.
  3. Click on "Create new secret key".
  4. Give it a name, create it, and copy the key immediately (you won't be able to see it again).
smart_toy

Anthropic Claude

  1. Visit the Anthropic Console at console.anthropic.com/settings/keys.
  2. Sign in or create an account.
  3. Click "Create Key".
  4. Copy the key for use in the application.
rocket_launch

xAI Grok

  1. Go to the xAI API Console at console.x.ai.
  2. Sign in to your account.
  3. Navigate to the API Keys section and create a new key.
terminal

Ollama

Ollama does not require an API key, but due to CORS restrictions on web applications, you must use a remote or publicly exposed instance (e.g. via ngrok) rather than localhost.

  1. Host Ollama remotely or expose your local instance securely using tools like ngrok.
  2. Ensure the instance allows Cross-Origin Resource Sharing (CORS) from this domain.
  3. In CortexPDF, leave the API key blank and enter your remote URL (e.g. https://your-ngrok-url.app) as the Base URL.
security

Security Reminder: API keys you enter in CortexPDF are stored only in your browser's memory and are never sent to our servers or saved permanently.
