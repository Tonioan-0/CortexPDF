# CortexPDF — Python Utilities (Legacy / Experimental)

> **Note:** CortexPDF has been rewritten as a **fully static, serverless web application**. The primary application code now lives in the `../Site/` directory.

The `App/` directory contains legacy Python backend utilities and experimental scripts from earlier versions of CortexPDF, which relied on FastAPI and PyMuPDF.

If you are looking to run or develop CortexPDF, please refer to the main [README.md](../README.md) in the root directory.
